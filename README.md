# Rock Paper Scissors Game:-

A simple and interactive Rock Paper Scissors game built using HTML, CSS and JavaScript.

## How to Play:-

1. Choose Rock, Paper, or Scissors by clicking on the image.
2. The computer will automatically make its choice.
3. The winner is decided according to these rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. Scores are updated after every round.
5. If both choices are the same, the game ends in a draw.

## Features:-

* User vs Computer gameplay
* Random computer choice generation
* Real-time score tracking
* Winner and draw messages
* Simple and attractive user interface

## Technologies Used:-

* HTML
* CSS
* JavaScript
